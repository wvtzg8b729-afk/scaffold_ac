Scaffold AC

Samlet udgangspunkt til anti-cheat–relateret arbejde: webpanel (Next.js), API og FiveM-resource i ét repo — med tydelig struktur og dokumentation til produktion.

Klon / fork repo Læs licens

Denne side er statisk HTML+CSS (ingen eksterne stylesheets). Panelet under web/ hostes separat med Node.

Oversigt

Hvad er det?

Et staff-orienteret panel til servere, bans og konfiguration, plus en Lua-resource der kan synkronisere mod dit panel. Målet er en ramme du kan bygge videre på — ikke en klon af kommercielle produkter.

Panel

Next.js kræver Node

Mappen web/ har API-ruter og Prisma. Den kan ikke køre som ren GitHub Pages — brug fx Vercel, Railway eller egen VPS.

cd web
npm install
npm run dev

Miljø: web/.env.example
