Scaffold AC

Et samlet udgangspunkt til anti-cheat-relateret arbejde på FiveM: webpanel (Next.js), API og FiveM-resource i samme repo — med fokus på klar struktur og driftsklar opsætning.

Hvad er det?

Scaffold AC samler et moderne admin-panel, server-side logik og resource-kode, så du kan konfigurere bans, servere og integrationer ét sted. Denne side er en statisk oversigt; selve panelet kører som en separat Node-applikation.

Kildekode på GitHub (repo: scaffold_ac) →

Panel (Next.js) kræver Node-hosting

Mappen web/ indeholder Next.js-appen med API routes og Prisma. Den kan ikke køre som ren statisk GitHub Pages — du skal hoste den på f.eks. Vercel, Railway, egen VPS eller lignende med Node og database.

Lokalt fra repo-roden:

  • cd web — gå ind i panel-projektet
  • npm install — hent afhængigheder
  • npm run dev — start udviklingsserver

Se web/.env.example for miljøvariabler og Prisma/SQLite eller anden database efter behov.

FiveM-resource

Resource-kode findes under fivem-resource/ (fx scaffold_ac). Følg respektive INSTALL.txt-filer i resource-mappen for server.cfg og afhængigheder.
